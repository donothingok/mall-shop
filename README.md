# Mall Shop Scaffold

商城系统基础脚手架（前后端分离）：

- `mall-backend`: Spring Boot 后端接口（模块化单体）
- `mall-web-store`: Vue3 前台 H5
- `mall-admin`: Vue3 管理后台
- `mall-mini`: uni-app 小程序
- `docs`: 设计与接口文档

## 快速开始

### 1) 后端
```bash
cd mall-backend
mvn spring-boot:run
```

### 2) 前台 H5
```bash
cd mall-web-store
npm install
npm run dev
```

### 3) 管理后台
```bash
cd mall-admin
npm install
npm run dev
```

### 4) uni-app
使用 HBuilderX 或 `@dcloudio/uni-app` CLI 打开 `mall-mini`。
